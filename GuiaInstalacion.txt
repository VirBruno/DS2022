# DS Travel - GR07
***
Sistema para búsqueda y venta de Pasajes Aéreos, Paquetes Turísticos,  Alojamiento y Vehículos.
En este caso, el sistema esta acotado a la gestion de Pasajes Aéreos y todo lo relacionado para que el sistema funcione.

### Info General 
***
El sistema permite la optimización del proceso de venta, la sistematización y agilizar estas actividades para la comodidad de los clientes
al momento de realizar un reserva online. 
El sistema consta de 2 módulos: administrativo (backoffice) y de usuarios finales (frontoffice).

## Installation
***
Para instalar el proyecto se requiere tener los siguientes ejecutables de la tecnologías mencinadas descargados en la computadora. 

Se debe copiar el link del repositorio de github: https://github.com/VirBruno/DS2022.git

Se deberá instalar MySQL, corriendo los scripts de creación y generación de datos:
1. DS Travel.sql
2. DS Travel registros.sql

Instalar PHP, Composer y NodeJS. 

Luego de chequear lo anteriormente dicho, abrir una ventana de comandos (se sugiere terminal de VS Code). 

INSTALAR ANGULAR
```
$ git clone https://github.com/VirBruno/DS2022.git
$ cd ..\DS2022
$ npm install
$ npm run start
```

INSTALAR LARAVEL
```
$ composer global require "laravel/installer *Instalar laravel*
$ create-project laravel/laravel API *Creación de proyecto API*
```

Para abrir la aplicación web se deberá presionar en el enlace generado (4020)
